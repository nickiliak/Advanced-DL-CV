---
description: Generate a weekly report for an exercise
---

Generate `week_report/README.md` and a 1-page `.tex` file for the specified exercise.

1. Read the exercise PDF from `ExerciseX.Y/`
2. Analyze all code in `ExerciseX.Y/src/`
3. Include images from `ExerciseX.Y/outputs/` if they exist
4. Answer all questions from the PDF
5. Summarize code logic, results, and insights
6. Write both `ExerciseX.Y/week_report/README.md` and a `.tex` file

If no exercise is specified, ask the user which exercise to generate for.
